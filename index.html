Tag h1 - h6

As tags h1 até h6 são usadas para definir cabeçalho em HTML. A h1 define o maior cabeçalho, diminuindo gradativamente o tamanho até a h6.

Tag p

A tag p define um parágrafo de texto.

< índice

Tag mark

A tag mark serve para sublinhar ou destacar alguma palavra, como se fosse um marca texto, para enfatizar alguma informação.

< índice

Tag small

A tag small diminui o tamanho de uma palavra.

< índice

Tag i

A tag i define uma parte do texto em uma voz ou mood alternativo, transformando no formato itálico (levemente inclinada pra direita). Usado pra indicar um termo, frase, pensamento, etc.

< índice

Tag u

A tag u adiciona uma linha abaixo da palavra. Vem do termo underline.

< índice

Tag strong

A tag strong transforma uma palavra em negrito. Usado quando se quer destacar uma palavra ou parte importante do texto.

< índice

Tag ol, ul, li

A tag ol significa ordered list. É caracterizada por ser uma lista numérica ou alfabética. Já a tag ul (unordered list) é uma lista que não contem alguma ordem específica. A tag li é usada junto com as duas tags para definir cada item das listas.

< índice

Tag a

A tag a define um hyperlink. É usada para linkar uma página à outra. Um importante atributo dessa tag é href, que indica o destino do link.

< índice

Tag hr

A tag hr define uma quebra de página no HTML. Uma linha horizontal usada para separar conteúdos em uma página.

< índice

Tag sub e sup

A tag sub define um texto subscrito, ou seja, escrito meio caractere abaixo da linha, normalmente em uma fonte menor, usado geralmente em fórmulas química como H2O. A tag sup é caracterizada por colocar meio caractere acima da linha, usado geralmente para adicionar rodas de rodapé1.

< índice

Tag blockquote

A tag blockquote é utilizada para demarcar um parágrafo de uma citação. Usa-se a tag q q para citar apenas uma linha.
